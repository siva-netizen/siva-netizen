### About Me
AI engineer and storyteller of resilient LLM systems. During my 6-month rotation at KLA as a Software Quality Engineer (AI Development), I:
- Built an AI Test Case Copilot on a multimodal RAG stack that automated coverage expansion and cut manual effort by 40%.
- Architected high-volume data pipelines with multithreading, deduplication, automated backups, and RAGAS evaluation, shrinking ingestion latency by 3× and eliminating 95% dupe records.
- Delivered an autonomous LangGraph + MCP-Playwright testing agent that executed 50+ end-to-end scenarios per day, slicing regression cycles from hours to minutes.

Now I split time between research-y projects like an MQA→MLA architecture packager for faster long-context inference and shipping usable assistants such as “Echo,” a Windows voice agent powered by Gemini Live and Computer-MCP. My north star: design evaluation-first AI that gives teams superpowers while staying transparent and auditable.
